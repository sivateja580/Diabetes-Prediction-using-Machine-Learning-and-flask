Diabetes Prediction using Machine Learning & Flask

This project is a **Machine Learning + Flask Web Application** that predicts whether a person is **Diabetic or Not Diabetic** based on health-related input features.  
It uses the **Pima Indians Diabetes Dataset** and provides a beautiful animated UI for user interaction.

---

## Features

- ✔ Logistic Regression Machine Learning Model  
- ✔ Scaler (StandardScaler) pre-processing  
- ✔ Flask Web Framework  
- ✔ HTML + CSS (Fully animated, professional UI)  
- ✔ Two separate pages:
  - `index.html` → Input Form  
  - `result.html` → Shows Prediction Result  
- ✔ Green (Not Diabetic) & Red (Diabetic) Animated Result Card  
- ✔ GitHub-ready project structure  
- ✔ Works on Anaconda, PyCharm, VS Code, or terminal
